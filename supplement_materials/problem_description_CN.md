# 考虑时效限制的固化装箱调度问题（无装配工序）



> 基于 Hao et al. (2022) EJOR 模型修改：

> - 去除装配工序，固化后的物料即为最终产品；

> - 增加 WIP（待固化物料）的时效限制：每个物料有最晚必须加工（进罐）时间，超过该时间则报废；

> - **显式报废决策**：每个物料在 deadline 前必须“进罐加工”或“报废”，用决策变量 $$R_i$$ 刻画。



---



## 1. 问题描述



### 1.1 背景与生产流程



生产系统仅包含**固化（装箱）阶段**，取消后续装配工序。固化后的物料直接作为最终产品满足外部需求。



**固化（装箱过程）**  

各 WIP（待固化物料）放置于专用托盘上，送入不同长度的工业热压罐进行固化。同一热压罐内只能处理具有**相同固化配置**的物料。热压罐选定配置并完成 setup 后，须持续运行对应时长。



**时效限制与显式报废**  

每个物料 $$i$$ 在计划期初有 $$w_i$$ 单位待固化 WIP，并给定**最晚进罐时段** $$D_i$$。在 $$t \le D_i$$ 内，WIP 要么被投入热压罐（$$X_{imt}$$），要么在 $$D_i$$ 结束时确认为报废（$$R_i$$）。**不允许**在 $$t > D_i$$ 再投入加工；未在 deadline 前安排进罐的那部分 WIP，必须通过报废变量 $$R_i$$ 显式计入。



由于固化阶段是配置相关的装箱过程，且各 WIP 的批量计划需同时确定，该问题可归结为**带时效限制与显式报废的配置相关装箱调度问题**，核心特征：



- 热压罐**可重配置**；

- 加工时间**依赖配置**；

- 各 WIP 具有**硬时效窗口** $$D_i$$，且 WIP 在 deadline 前**加工/报废二选一**（物料守恒）；

- 装箱决策直接服务于最终需求（无装配拉动）。



### 1.2 基本假设



1. 所有输入参数已知，为**确定性**问题。

2. 瓶颈仅在固化阶段。

3. 最终产品（固化后的 WIP）外部需求允许**延期交货（backorder）**。

4. 固化过程有**提前期** $$l_{ti}$$：时段 $$t$$ 的投产量 $$X_{imt}$$ 在时段 $$t + l_{ti}$$ 到达成品流。

5. 托盘单行排列，容量由累计长度决定。

6. 各尺寸托盘数量充足。

7. 每个物料 $$i$$ 的 WIP 总量为 $$w_i$$；在 $$D_i$$ 之后不可再进罐，未加工部分由 $$R_i$$ 报废，不再产生有效产出。

8. 报废量 $$R_i$$ 与延期量 $$S_{it}^-$$ 含义不同：$$R_i$$ 记录**未及时进罐而丢弃的 WIP**；$$S_{it}^-$$ 记录**成品需求未被满足**的缺口（可能由产能不足、排程决策或报废导致）。

9. **$$sc_i > bc_i$$**：在可行排程存在时，模型应优先通过加工/缺货满足需求，而非轻易报废 WIP；仅当 deadline 与产能使 (9) 无法在不报废的情况下成立时，才取 $$R_i > 0$$。

## 2. 符号说明



### 2.1 集合与下标



| 符号 | 含义 |

|------|------|

| $$T$$ | 计划期内时段集合，下标 $$t$$ |

| $$I$$ | WIP（待固化物料 / 最终产品）集合，下标 $$i$$ |

| $$U$$ | 固化配置集合，下标 $$u$$ |

| $$M$$ | 热压罐集合，下标 $$m$$ |

| $$T_i^{\le D} = \{ t \in T : t \le D_i \}$$ | 物料 $$i$$ 允许进罐的时段集合 |



### 2.2 参数



| 符号 | 含义 |

|------|------|

| $$l_{ti}$$ | 物料 $$i$$ 的固化提前期 |

| $$b_{iu}$$ | 若物料 $$i$$ 需在配置 $$u$$ 下固化则为 1，否则为 0 |

| $$l_u$$ | 配置 $$u$$ 的固化持续时长 |

| $$d_{it}$$ | 时段 $$t$$ 物料 $$i$$ 的外部需求 |

| $$w_i$$ | 物料 $$i$$ 的**初始 WIP 量**（计划期初待固化、必须在 $$D_i$$ 前加工或报废） |

| $$D_i$$ | 物料 $$i$$ 的**最晚进罐时段**（仅允许 $$t \le D_i$$ 投入 $$X$$） |

| $$v_i$$ | 物料 $$i$$ 的托盘长度 |

| $$q_m$$ | 热压罐 $$m$$ 的最大累计托盘长度 |

| $$p_{mcu}$$ | 热压罐 $$m$$ 在配置 $$u$$ 下、时段 $$t$$ 发生 setup 的生产成本（简记可用 $$p_{cum}$$） |

| $$h_{ci}$$ | 物料 $$i$$ 的单位 holding 成本 |

| $$bc_i$$ | 物料 $$i$$ 的单位 backorder 成本 |

| $$sc_i$$ | 物料 $$i$$ 的单位**报废成本**（须满足 $$sc_i > bc_i$$，使报废劣于缺货） |



**参数 $$w_i$$ 的默认取法（算例生成建议）**  

$$w_i = \sum_{t \in T} d_{it}$$  

即：每个物料释放的 WIP 量等于计划期内总需求量。若实际批次投产量与总需求不同，可在算例 JSON 中单独给定 $$w_i$$。

**参数 $$sc_i$$ 的默认取法（算例生成建议）**  

$$sc_i = \lambda \cdot bc_i, \quad \lambda > 1$$  

默认 $$\lambda = 2$$（代码中 `SCRAP_COST_MULTIPLIER = 2.0`，即报废成本为缺货成本的 2 倍）。若算例 JSON 提供 `scrap_costs`，须逐 item 校验 $$sc_i > bc_i$$。



### 2.3 决策变量



| 符号 | 含义 |

|------|------|

| $$S_{it}^+$$ | 时段 $$t$$ 末物料 $$i$$ 的**成品**库存量 |

| $$S_{it}^-$$ | 时段 $$t$$ 末物料 $$i$$ 的延期量（backorder） |

| $$X_{imt}$$ | 时段 $$t$$ 投入热压罐 $$m$$ 的物料 $$i$$ 数量（进罐量） |

| $$Y_{umt}$$ | 时段 $$t$$ 热压罐 $$m$$ 为配置 $$u$$ 做 setup（1/0） |

| $$Z_{umt}$$ | 时段 $$t$$ 热压罐 $$m$$ 在配置 $$u$$ 下运行（1/0） |

| $$R_i$$ | 物料 $$i$$ 的**报废量**（在 $$D_i$$ 前未进罐、因而被丢弃的 WIP 数量） |

## 3. 数学模型



单阶段固化调度 + 时效限制 + 显式报废。以下为集成模型（IM）。



### 3.1 目标函数



$$

\min \sum_{t \in T} \sum_{i \in I} h_{ci} S_{it}^+

+ \sum_{t \in T} \sum_{i \in I} bc_i S_{it}^-

+ \sum_{t \in T} \sum_{m \in M} \sum_{u \in U} p_{mcu} Y_{umt}

+ \sum_{i \in I} sc_i R_i

\tag{1}

$$



前三项分别为 holding、backorder、热压罐 setup/生产成本；最后一项为**报废成本**。



### 3.2 主要约束



**（2）成品库存平衡**（到达量为提前 $$l_{ti}$$ 期前的进罐量之和）：



$$

S_{i,t-1}^+ - S_{i,t-1}^- + \sum_{m \in M} X_{i m,\, t - l_{ti}}

= d_{it} + S_{it}^+ - S_{it}^-

\quad \forall i \in I,\; t \in T

$$



（约定 $$S_{i0}^+ = S_{i0}^- = 0$$；若 $$t - l_{ti} < 1$$，对应 $$X$$ 项为 0。）



**（3）–（5）热压罐配置约束**（与原模型相同）：



$$

\sum_{u \in U} Y_{umt} \leq 1 \quad \forall m \in M,\; t \in T

\tag{3}

$$



$$

\sum_{u \in U} Z_{umt} \leq 1 \quad \forall m \in M,\; t \in T

\tag{4}

$$



$$

\sum_{t'=\max(t-l_u+1,\,1)}^{t} Y_{u m t'} = Z_{umt} \quad \forall u,m,t

\tag{5}

$$



**（6）–（7）装箱容量约束**：



$$

X_{imt} \leq \left\lfloor \frac{q_m}{v_i} \right\rfloor \cdot \left( \sum_{u \in U} b_{iu} Y_{umt} \right) \quad \forall i,m,t

\tag{6}

$$



$$

\sum_{i \in I} v_i X_{imt} \leq q_m \quad \forall m,t

\tag{7}

$$



**（8）时效进罐窗口**（禁止超期进罐）：



$$

X_{imt} = 0 \quad \forall i \in I,\; m \in M,\; t > D_i

\tag{8}

$$



**（9）WIP 物料守恒（显式报废）** — **新增核心约束**：



$$

\sum_{m \in M} \sum_{t \in T_i^{\le D}} X_{imt} + R_i = w_i

\quad \forall i \in I

\tag{9}

$$



含义：每个物料的初始 WIP $$w_i$$，必须在 deadline 前全部**进罐**或**报废**，不允许“悬空”。  

由 (8)(9) 可知：若因产能或排程原因无法将全部 $$w_i$$ 在 $$t \le D_i$$ 投入，差额必须由 $$R_i > 0$$ 显式吸收；相应可产出的成品减少，缺口通过 (2) 体现为 $$S_{it}^-$$ 上升。



**（9）与旧版隐式报废的关系**  

旧版仅用 (8) 禁止超期 $$X$$，未追踪“有多少 WIP 被报废”；新版 (9) 把**未进罐的 WIP 量**显式分配给 $$R_i$$，便于在 LRMP/B&P 中分支与分析。



### 3.3 变量域



$$

S_{it}^+,\ S_{it}^- \ge 0;\quad

R_i \ge 0;\quad

X_{imt} \in \mathbb{Z}_+;\quad

Y_{umt},\ Z_{umt} \in \{0,1\}

$$



（LRMP 松弛时：$$X$$、$$R_i$$ 可松弛为连续；$$Y$$、$$Z$$ 或 $$Q$$ 列权重按列生成框架处理。）



### 3.4 模型逻辑小结



| 变量/约束 | 作用 |

|-----------|------|

| $$X_{imt}$$，$$t \le D_i$$ | 在时效窗口内安排进罐 |

| (8) | 超期不可再投 |

| (9) + $$R_i$$ | WIP 要么进罐要么报废，物料守恒 |

| (2) + $$S_{it}^-$$ | 成品侧需求缺口（backorder） |

| $$sc_i R_i$$ | 惩罚报废；$$sc_i > bc_i$$ 保证“能加工则不报废”，仅 deadline/产能不可行时 $$R_i > 0$$ |



**典型因果链**  

产能/时间不足 → 无法在 $$D_i$$ 前投够 $$w_i$$ → $$R_i > 0$$ → 可到达成品的 $$X$$ 总和下降 → $$S_{it}^-$$ 增加 → backorder 成本上升。

在 Dantzig-Wolfe 分解中，每台机器 $$m$$ 的列仍表示可行调度方案；主问题除原有 $$Q$$、$$S^+$$、$$S^-$$ 外，**增加标量报废变量 $$R_i$$** 及约束 (9)：



$$

\sum_{m \in M} \sum_{k \in K_m} \left( \sum_{t \in T_i^{\le D}} x_{imt}^k \right) Q_{mk} + R_i = w_i

\quad \forall i

$$



其中 $$x_{imt}^k$$ 为列 $$k$$ 在 $$(i,m,t)$$ 上的进罐量。  

$$R_i$$ 保留在主问题（不进入子问题），对偶变量 $$\rho_i$$ 进入定价子问题，影响 reduced cost。
